## Typed errors (3 -- schema layer)

- `VISUAL_FIELDS_REQUIRED` -- raised when an action tool is missing `visual_reason` or `client`.
- `BADGE_NOT_ALLOWED` -- raised when `client` is not on the v0.9.36 shared allowlist. Allowed values: Claude, Codex, ChatGPT, Perplexity, Windsurf, Cursor, Antigravity, OpenCode, OpenClaw, Grok, Gemini, Hermes (added v0.9.69 / PR #49).
- `TOOL_REMOVED` -- raised when callers invoke the removed `start_visual_session` / `end_visual_session` tools. The contract is now implicit.
