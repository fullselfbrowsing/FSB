<objective>
Fix two tab-scoping bugs in the FSB Chrome extension sidepanel surface:

1. **Panel visibility leaks across tabs.** The sidepanel currently stays enabled on every tab. We add a `chrome.tabs.onActivated` listener in `extension/background.js` that calls `chrome.sidePanel.setOptions({ tabId, enabled: <bool> })` per the tab's working state. Working tabs (a tab with an active automation session) keep the panel enabled; non-working tabs collapse it. A `chrome.action.onClicked` enable-before-open path lets the user force-open the panel with welcome state on idle tabs.

2. **Completion message routes to the wrong tab.** When `automationComplete` fires for a session dispatched from tab A, the sidepanel currently renders the completion AND persists it via `_persistMessage` using the MODULE-SCOPE `conversationId` (which has been swapped to tab B's conv on tab switch). The completion bubble lands in tab B's persisted log. Fix: route the completion to `request.conversationId` (already on the broadcast payload from `background.js`), persisting into the originating conversation regardless of which tab is currently displayed. The DOM render only fires when the message's conv matches the active conv; otherwise persist silently so the next `swapToTabConversation` -> `hydrateChatFromConversationId` replay shows it.

Purpose: Per-tab panel visibility matches "has work running here"; completion messages land in the dispatching tab's conversation regardless of focus.

Output: Three file edits (background.js, sidepanel.js, new test) plus one package.json registration. No build step required (background.js is the classic SW entry; sidepanel.js is `<script src>` loaded; neither is in esbuild ENTRIES).
</objective>

```js
/**
 * QT-7bi-02 (completion-routing fix) -- explicit-convId variant of
 * _persistMessage.
 *
 * The original _persistMessage closes over the module-scope `conversationId`
 * variable, which is mutated by swapToTabConversation on every tab switch.
 * When automationComplete fires for a session dispatched from tab A while
 * the sidepanel currently displays tab B's conversation, _persistMessage
 * would write the completion bubble into tab B's persisted log (the
 * currently-displayed conv), not tab A's (the originating conv).
 *
 * This sibling helper takes an explicit `convId` so completion-routing
 * call sites can persist into the originating conversation regardless of
 * which tab is currently displayed. Identical guards + buffer + debouncer
 * semantics as _persistMessage.
 *
 * Guards:
 *  - convId must be a non-empty string (lazy-mint windows pass null; skip).
 *  - sidecar absent: skip (script-tag load order failure).
 *  - debouncer absent: skip (boot init failed; storage write unsafe).
 *
 * Storage failures swallow silently -- DOM render must never block on
 * persistence (mirrors _persistMessage contract).
 */
function _persistMessageToConversation(role, content, kind, convId) {
  if (typeof FSBSidepanelMessageLog === 'undefined') return;
  if (!convId || typeof convId !== 'string') return;
  if (typeof content !== 'string' || content.length === 0) return;
  if (!_messageLogDebouncer) return;

  var resolvedRole = (role === 'user') ? 'user' : 'assistant';
  var resolvedKind = (typeof kind === 'string' && kind.length > 0) ? kind : 'text';

  var buffer = _messageLogPendingBuffer.get(convId);
  if (!buffer) {
    buffer = [];
    _messageLogPendingBuffer.set(convId, buffer);
  }
  buffer.push({
    role: resolvedRole,
    content: content,
    timestamp: Date.now(),
    kind: resolvedKind
  });

  _messageLogDebouncer.schedule(convId, function () {
    return _flushMessageLog(convId);
  });
}
```

```js
function extractAfterAnchor(src, anchor) {
  var startIdx = src.indexOf(anchor);
  if (startIdx === -1) return null;
  var i = startIdx + anchor.length - 1;
  if (src[i] !== '{') {
    var braceIdx = src.indexOf('{', i);
    if (braceIdx === -1) return null;
    i = braceIdx;
  }
  var depth = 1;
  i++;
  while (i < src.length && depth > 0) {
    var c = src[i];
    if (c === '{') depth++;
    else if (c === '}') depth--;
    if (depth === 0) break;
    i++;
  }
  if (depth !== 0) return null;
  return src.slice(src.indexOf('{', startIdx) + 1, i);
}
```
