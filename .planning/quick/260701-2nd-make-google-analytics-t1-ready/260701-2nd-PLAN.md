# Make Google Analytics T1 Ready

## Scope

Promote the imported Google Analytics `ganalytics.*` read descriptors to a conservative T1a execution path:

- Add a bundled `ganalytics` head handler for all eight read-only Google Analytics descriptors on `https://analytics.google.com`.
- Route execution through the bounded page-read primitive, where the injected page function calls the page-owned `gapi.client.request` without returning or logging credential material.
- Register the head in service-worker loading, catalog seeding, search readiness, readiness reporting, port-contract verification, and origin classification.
- Add focused tests proving read-only handler registration, page-read dispatch, GAPI request construction, fail-closed behavior, and readiness promotion.
