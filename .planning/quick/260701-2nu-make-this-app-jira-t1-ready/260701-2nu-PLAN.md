# Quick 260701-2nu Plan: Jira T1 Readiness

## Objective

Make the existing `jira.*` catalog descriptors T1-ready for Atlassian Jira Cloud tenants: safe REST reads become reviewed T1a handlers pinned to an active `*.atlassian.net` tab, while mutations remain guarded fail-closed until live mutation-body UAT exists.

## Scope

T1-ready reads: `jira.get_issue`, `jira.get_myself`, `jira.get_project`, `jira.get_transitions`, `jira.list_boards`, `jira.list_comments`, `jira.list_issue_types`, `jira.list_priorities`, `jira.list_projects`, `jira.list_sprints`, `jira.search_issues`, `jira.search_users`.

Guarded fail-closed rows: `jira.add_comment`, `jira.add_watcher`, `jira.assign_issue`, `jira.create_issue`, `jira.delete_issue`, `jira.link_issues`, `jira.transition_issue`, `jira.update_issue`.
