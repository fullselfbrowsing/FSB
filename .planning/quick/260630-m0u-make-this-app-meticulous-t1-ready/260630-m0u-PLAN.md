# Make Meticulous T1-ready where safe

## Objective

Promote the safe Meticulous same-origin read surface from discovery-pending to T1a by adding a reviewed app-specific handler for `https://app.meticulous.ai/api/graphql`, while leaving all Meticulous write/mutation descriptors non-executable until live mutation-body UAT and write evidence exist.

## Source Coverage Audit

- `.planning/STATE.md` and `.planning/PROJECT.md`: preserve the two-tool surface (`search_capabilities`, `invoke_capability`), MV3 data/code wall, same-origin proof standard, and fail-closed writes.
- `51-T1-TAIL-WORKLIST.md/json`: cover the 21 Meticulous `sideEffectClass: read` same-origin candidates; keep the 5 Meticulous `write-destructive-uat` rows out of the handler.
- Existing quick patterns: follow the recent `shortcut`, `leetcode`, `wikipedia`, `hackernews`, `npm`, `yelp`, `tripadvisor`, and `zillow` handler pattern: reviewed bundled code, `ctx.executeBoundSpec` only, origin pin, shape guards, no direct fetch/chrome APIs, report refresh.
- Dirty worktree: treat current modified/untracked files as user or parallel-agent work. Do not revert or normalize unrelated changes.

## Safe Read Slugs

Register only these Meticulous read descriptors when their GraphQL operation is a query and the handler test proves the expected shape:

`meticulous.get_current_user`, `meticulous.get_project`, `meticulous.get_project_pull_request`, `meticulous.get_replay`, `meticulous.get_replay_screenshots`, `meticulous.get_session`, `meticulous.get_session_events`, `meticulous.get_test_run`, `meticulous.get_test_run_coverage`, `meticulous.get_test_run_diffs`, `meticulous.get_test_run_pr_description`, `meticulous.get_test_run_screenshots`, `meticulous.get_test_run_source_code`, `meticulous.get_test_run_test_cases`, `meticulous.list_github_repositories`, `meticulous.list_organization_members`, `meticulous.list_organizations`, `meticulous.list_projects`, `meticulous.list_replays`, `meticulous.list_sessions`, `meticulous.search_sessions`.

Do not register these Meticulous write rows: `meticulous.accept_all_diffs`, `meticulous.check_for_flakes`, `meticulous.compare_replays`, `meticulous.create_label_action`, `meticulous.upsert_diff_approval`.
