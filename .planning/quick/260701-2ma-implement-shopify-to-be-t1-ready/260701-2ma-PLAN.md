# Quick 260701-2ma Plan: Shopify T1 Readiness

## Objective

Make the existing `shopify.*` catalog descriptors terminal-state T1-ready for Shopify Admin: read descriptors become reviewed `T1a` same-origin handlers, and payment/destructive mutations stay guarded fail-closed until live mutation-body UAT exists.

## Scope

T1-ready reads: `shopify.list_products`, `shopify.get_product`, `shopify.list_orders`.

Guarded fail-closed rows: `shopify.create_order`, `shopify.cancel_order`.
