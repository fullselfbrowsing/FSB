# Quick Task 260701-2nw: Implement TikTok T1 Readiness

## Scope

Make the existing TikTok catalog rows terminal and truthful:

- Promote proven public same-origin SSR reads to T1a:
  - `tiktok.get_user_profile`
  - `tiktok.get_video`
- Keep private/session-bound or signed API rows guarded fail-closed:
  - `tiktok.get_current_user`
  - `tiktok.get_followers`
  - `tiktok.get_following`
  - `tiktok.get_for_you_feed`
  - `tiktok.get_notifications`
  - `tiktok.search_users`
  - `tiktok.search_videos`

Do not introduce cross-origin API fetches, direct `fetch`, `chrome.tabs`, `chrome.scripting`, cookie/storage reads, bearer auth, signed TikTok API assumptions, or secret-bearing logs.
