# Plan 34-01 — `upload_file` Tool

Single plan. A new first-class action tool that sets a real file from a disk path onto an `<input type=file>` via CDP `DOM.setFileInputFiles`, threaded through both front doors with a shared security chokepoint.

## Task 1 — Denylist module (UPLOAD-03)
`extension/utils/upload-path-denylist.js`: pure string sensitive-path policy (dirs: `~/.ssh`, `~/.aws`, `~/.gnupg`, keychains, `/var/run/secrets`, FSB vault; basenames: `id_*`, `.env*`, `credentials`, `.git-credentials`, history files; suffixes: `.pem/.key/.p12/...`; abs prefixes: `/etc/shadow`, `/proc/`). `classify`/`isDenied`/`isAbsolutePath`/`basenameOf`, IIFE dual-export `globalThis.FsbUploadPathDenylist`. Win32 trailing-dot/space stripped so they cannot bypass. `importScripts` in background.js.

## Task 2 — Shared background CDP helper (UPLOAD-01, UPLOAD-03)
`extension/background.js` `executeUploadFile(tabId, selector, filePath)` beside `executeCDPToolDirect`: input + absolute-path validation → denylist check (block + audit, BEFORE any side effect) → `chrome.debugger.attach('1.3')` (stale-retry) → `DOM.getDocument`→`DOM.querySelector`→`DOM.describeNode` (is `INPUT[type=file]`?)→descendant `input[type=file]` fallback→`DOM.setFileInputFiles{nodeId,files:[path]}` → detach in `finally` → audit via `FsbAuditLog.append` (origin + outcome + decision only; never the path). Mirrors the existing CDP attach/detach seam.

## Task 3 — Two front doors → one chokepoint (UPLOAD-02)
- MCP: `extension/ws/mcp-tool-dispatcher.js` `MCP_PHASE199_TOOL_ROUTES.upload_file` + `handleUploadFileRoute` (validates params, calls `globalThis.executeUploadFile`; tab ownership already enforced by the gate).
- Autopilot: `extension/ai/tool-executor.js` `case 'upload_file':` in `executeBackgroundTool` (calls the same helper).
- `extension/ai/tool-definitions.js`: `upload_file` registry entry (`_route:'background'`, `_readOnly:false`, `selector`+`file_path` required, multi-agent boilerplate). `mcp/ai/tool-definitions.cjs` regenerated byte-identical via `npm --prefix mcp run build`.
