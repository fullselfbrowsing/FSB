# 48-01 Plan: Add Vercel Same-Origin Read Head

## Goal

Add a narrow Vercel T1a handler for non-secret GET reads under `https://vercel.com/api`.

## Scope

- Add `catalog/handlers/vercel.js`.
- Mirror it to `extension/catalog/handlers/vercel.js`.
- Port seven read slugs: current user, teams, projects, deployments, and domains.
- Keep `vercel.list_env_vars` out of the active batch.

## Verification

- Handler source has no direct network call, no `chrome.scripting`/`chrome.tabs`, and no secret-bearing console logging.
