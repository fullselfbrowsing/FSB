| Boundary | Description |
|----------|-------------|
| content script -> service worker (value report) | The untrusted page text crosses here in `value.text`. The existing sender guard (:5438) rejects foreign senders; the new case type-validates and length-caps before persisting. This is the primary Phase-16 threat surface. |
