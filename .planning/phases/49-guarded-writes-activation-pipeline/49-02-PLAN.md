# 49-02 Plan: Evidence Verification Gate

## Goal

Make write activation evidence machine-checkable and wire it into normal extension validation.

## Scope

- Add `scripts/verify-write-activation-evidence.mjs`.
- Add `tests/write-activation-evidence.test.js`.
- Append the verifier to `npm run validate:extension`.

## Verification

- The committed ledger passes.
- A missing active-write evidence record fails.
- A missing guarded-write record fails.
- Marking a guarded write as active fails.
- Secret-like literals in evidence fail.
