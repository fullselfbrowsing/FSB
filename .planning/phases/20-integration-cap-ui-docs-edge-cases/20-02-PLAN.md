| chrome.storage.session trigger registry -> background conflict scan | Persisted snapshots may be stale, malformed, terminal, or owned by another caller. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-20-02-01 | Information Disclosure | `fsbTriggerHandleToolArm` conflict response | mitigate | Resolve owner context and access denial before conflict scan; return only minimal conflict metadata after authorization. |
| T-20-02-02 | Tampering | `fsbTriggerFindTabWatchConflict` registry scan | mitigate | Ignore malformed records and normalize watch mode; never trust arbitrary registry shape. |
| T-20-02-03 | Denial of Service | Cross-mode same-tab arming | mitigate | Reject before read, persistence, observer startup, and pulse startup so conflicting requests cannot consume cap slots or start watchers. |
