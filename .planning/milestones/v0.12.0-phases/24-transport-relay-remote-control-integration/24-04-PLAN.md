# 24-04 Plan: Remote-Control Parity

## Goal

Align dashboard reverse-control frames with PhantomStream remote-control helpers while preserving FSB's CDP dispatch, viewport mapping, retargeting, and user-visible ownership/debugger states.

## Tasks

1. Map legacy dashboard remote-control messages to PhantomStream `REMOTE_CONTROL` helpers without breaking current static or Angular dashboards.
2. Validate click/type/scroll payloads with package helpers before CDP dispatch where compatible.
3. Preserve current FSB ownership state:
   - attached;
   - blocked by external debugger;
   - retarget-required;
   - user-stop;
   - no-tab/restricted-tab diagnostics.
4. Verify coordinate mapping uses the renderer/viewer viewport mapping and rejects stale frames/session mismatches.
5. Add tests covering stale frame rejection, stream-tab changes, navigation retargeting, debugger contention, and safe summaries for text input.
